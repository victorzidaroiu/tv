## TV Products App

This app allows you to select and order tv products. It is built using the MEAN stack and it implements a REST API to communicate with the AngularJS front end.

http://tv.zidaroiu.com

### Requirements

* MongoDB

### Installation

* ``` nvm use ```
* ```cp .env.dev .env```
* ``` npm i ```

### Start server

``` npm start ```

### Build

``` npm run build ```

### Test

``` npm test ```
