## TV Products App

A small web app using the MEAN stack and a RESTful API.

### Demo

http://tv.zidaroiu.herokuapps.com

### Requirements

- node
- MongoDB
- Mocha
- Grunt

- mongoDB

### Installation

``` nvm use ```
``` npm i ```

### Start server

``` npm start ```

### Build

``` npm run build ```

### Test

``` npm test ```
