## TV Products App

This app allows you to select and order tv products. It is built using the MEAN stack and it implements a RESTful API to communicate with the AngularJS front end.

http://tv.zidaroiu.com

### Requirements

- node
- MongoDB

### Installation

``` npm i -g mocha```
``` npm i -g grunt-cli```
``` nvm use ```
``` npm i ```

### Start server

``` npm start ```

### Build

``` npm run build ```

### Test

``` npm test ```
