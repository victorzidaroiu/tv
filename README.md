## TV Products App

This app allows you to order items. It is build using the MEAN stack and it implements RESTful API to communicate to the AngularJS front end.

http://tv.zidaroiu.com

### Requirements

- node
- MongoDB
- Mocha
- Grunt

### Installation

``` nvm use ```
``` npm i ```

### Start server

``` npm start ```

### Build

``` npm run build ```

### Test

``` npm test ```
